# Unity 原生3D物体

## 自带模型

Unity 自带以下几种模型：

- Cube 立方体
- Sphere 球体
- Capsule 胶囊体
- Cylinder 圆柱体
- Plane 平面

Unity尺寸单位约定为1米

## 注意事项

原生模型仅用于练习，真正的模型应该在建模软件里制作

CG建模软件：3DsMax/Maya/Blender/Cinema 4D/ZBrush
